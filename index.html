The name of a colour will appear and it will be filled in a different colour. You will be asked to select either the word itself or the colour of the word. Try to rack up as many points as you can in 20 seconds!

Example:

Select the color

RED

Here, the answer would be 'blue', since the actual colour of the word is blue. Similarly, if the task was to 'Select the text', the answer would be 'red'.
